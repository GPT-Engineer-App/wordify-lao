# wordify-lao

Develop an AI system that converts PDF documents to editable Word format, ensuring high accuracy OCR and layout preservation, with specialized support for the Lao language.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/wordify-lao.git
cd wordify-lao
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
